## 原理介绍

子域名枚举就是利用 **DNS 解析**，去测试某个可能存在的子域名是否能被 DNS 服务器正确解析。通常会准备一份常见子域名字典，

然后拼接主域名进行批量查询，如果查询 DNS 后返回了对应的 IP、CNAME 或其他有效记录，就说明这个子域名存在

**泛解析识别**

有些域名开启了泛解析，也就是：

- `aaa.example.com`
- `bbb.example.com`
- `123.example.com`

都会返回同一个 IP

所以子域名枚举里还要做一件事：判断是否存在泛解析，并过滤假结果

解决方案：随机生成几个不存在的名字去解析，如果也能返回结果，就说明可能开了泛解析
