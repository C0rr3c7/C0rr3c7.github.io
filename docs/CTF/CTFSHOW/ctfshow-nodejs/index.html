});

    .digest('hex');
}

> crypto.createHash('md5').update(a+'123').digest('hex');

['a']+flag= = ='a'+flag，比如flag是flag123，那么最后得到的都是aflag[123}，因此这个也肯定成立：md5(['a']+flag)= = =md5('a'+flag)，同时也满足a!==b

?a[0]=1&b[0]=2 不行

但是如果传a[0]=1&b[0]=2，相当于创了个变量a=[1] b=[2]，再像上面那样打印的时候，会打印出1flag{xxx}和2flag{xxx}，md5不相等

    for (let key in object2) {
